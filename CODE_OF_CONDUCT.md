# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct.

We expect everyone participating in the project — issues, pull requests,
discussions, and any other project space — to follow it.

## Reporting

Report unacceptable behaviour privately to **conduct@utilified.com**.
Reports are handled confidentially by the maintainers.

We aim to acknowledge reports within **5 business days**, matching the
response cadence in [SECURITY.md](./SECURITY.md).
